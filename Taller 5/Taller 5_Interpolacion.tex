\subsection{LAGRANGE}
Se consideran tres ejercicios en cuales se aplica los conocimientos adquiridos en clase acerca de la obtención de un polinomio de interpolación. Se hace uso de la base de Lagrange para el cual se necesita un conjunto de $n+1$ puntos en el plano, es decir distintas coordenadas en $x$: \\

\begin{displaymath}
(x_0,y_0),(x_1,y_1),(x_2,y_2),...,(x_n,y_n)
\end{displaymath}

El interés es encontrar una función polinómica que pase por esos $n+1$ puntos y que tenga el menor grado, este polinomio será el denominado polinomio de interpolación.\\ 

Así pues se hace uso de la fórmula general para el polinomio de interpolación de Lagrange:\\
\begin{equation}
P(x)=\sum_{i=0}^{n}l_i(x)\cdot y_i
\end{equation}
\\
Para ello son usados los polinomios básicos de Lagrange dados por: \\

\begin{equation}
l_i(x)=\prod_{\begin{matrix}
m=0\\ m\neq 0
\end{matrix}}^{n} \frac{(x-x_m)}{(x_i-x_m)}
\end{equation}

Al modificar el m-archivo $lagrange$ como $lagrange_2$ para que ahora el parametro $z$ sea un vector al igual que $valor$, donde $nz$ es el tamaño de $z$ y para que sea recorrido se utiliza un $for - end$ en MATLAB.

\subsection{ESTIMACIÓN DEL ERROR DE INTERPOLACIÓN}
Se utiliza la fórmula para el calculo del error abosulto:

\begin{equation}
\left  |f(x) - p_n(x)  \right |< 10^{-8}
\end{equation}
\textbf{donde}, \\
\textbf{$p_n(x)$}: es el polino de interpolación. \\
\textbf{$f(x)$}: la función dada.\\

Como resultado se obtuvo que elmenor grado posible para el polinomio de interpolación fue $15$ para la función $f(x)= senh(x)  para  x: [-1,1]$.

Considerando la función $g(x)=\frac{1}{1+25\cdot x²}$ se obtuvo para el apartado 'a' la grafica de la función y los polinomios que la interpolan. Que se podrá observar más adelante.\\

\subsection{FENÓMENO DE GIBBS}
Este fenómeno describe el comportamiento que tiene la serie de Fourier asociada a una función definida a trozos periódica en una discontinuidad evitable de salto finito.
Cuando la serie de Fourier presenta discontinuidades, no es posible una buena convergencia en sus entornos.


\subsection{SPLINE CÚBICO}

Se define como una curva diferenciable definida en partes mediante polinomios, este brinda resultados similares para la interpolación y solo requiere el uso de polinomios de bajo grado para evitar las oscilaciones. Para hallar el spline cúbico se utilizó la función "spline" de MATLAB para la evaluación numérica de los respectivos polinomios de interpolación.

\section{CONCLUSIONES}
\begin{itemize}
    \item El método de lagrange ha resultado ser efectivo en la obtención del polinomio de interpolación para una función, debido a que su algoritmo no utiliza muchos recursos computacionales debido a que sus iteraciones contienen operaciones matemáticas sencillas.\\
    \item El error máximo en todos los polinomios de interpolación que analizamos se encuentra siempre en los extremos del intervalo del dominio, sin importar que método se utilice.\\
    \item Los nodos de Chebyishev minimizan el problema del fenómeno de Runge, es decir, a medida que aumenta el grado polinomial se dismuye el error. Así se puede decir que los nodos de Chebyshef pueden resultar muy eficientes para la realización de un polinomio de interpolación.\\
    \item En el ejercicio 4 se observó que se cumple el fenómeno de Gibbs debido a que el error disminuye en el tercer decimal para el método de lagrange y el spline cúbico mostrando así que siempre va a existir un pico de error sin importar cuanto se aumente el grado polinomial.
    \item Se aprendió que el método del spline cúbico ha resultado ser el más eficiente sin importar que nodos se utilicen y el fenómeno que se esté probando. 
\end{itemize}

% Breve descripcion del programa: Crea un polinomio de interpolación para
% una función ______________________.
% Datos de entrada:
% ----------------
% x: Vector de nodos
% y: Vector de nodos
% z: Vector de valores a evaluar en polinomio de inteporlación
% Datos de salida:
% ---------------
% vector : Vector columna con los polinomios base de Lagrange
% -------------------------------------------------------------------------
function [valor] = lagrange_2(x,y,z)
n = length(x);
if length(y)~= n
    disp('Tamaño de  x e y deben ser el mismo');
    return
end
nz=length(z);
valor = zeros(nz,1);
for k=1:nz
for i = 1:n
    l = y(i);
    for j=1:n
        if i~= j 
            l = l*(z(k)-x(j))/(x(i)-x(j));            
        end
    end
    valor(k) = valor(k)+l;
end
end
